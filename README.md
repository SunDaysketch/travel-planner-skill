# 🌏 Travel Planner Skill for Claude.ai

> 一鍵生成互動式自由行旅遊網頁的 Claude Skill，支援全球城市、中英雙語切換、進階交通票種比價。

---

## ✨ 功能特色

- 🗺️ **全球城市支援** — 東京、首爾、大阪、巴黎、倫敦、台北…任何城市皆可
- 🚇 **進階交通攻略** — 自動比較 IC卡 vs Pass vs 散票，計算最划算方案
- ✈️ **機場接駁建議** — 各城市機場至市區路線費用 & 時間一覽
- 🌐 **中英雙語切換** — 網頁右上角一鍵切換，適合與外籍同伴共享
- 📍 **Google Maps 整合** — 每段路線、每個景點直連導航
- 📱 **行動裝置優先** — 手機瀏覽器直接開啟，零 App 安裝
- 📄 **單一 HTML 檔** — 零依賴，可離線使用，方便分享給同行者

---

## 🖥️ 網頁預覽

| 每日行程 | 交通攻略 | 購物清單 |
|---------|---------|---------|
| 彩色線路視覺化路線 | 票種費用比較表 | 依區域分組清單 |

## 📦 安裝方式

### Step 1：下載 Skill 檔

前往本 Repo 的 [Releases](../../releases) 頁面，下載最新版：

```
travel-planner.skill
```

### Step 2：上傳到 Claude.ai
1. 前往 [claude.ai](https://claude.ai) 並登入
2. 點擊右上角頭像 → **Settings**
3. 左側選單點 **Skills**
4. 點擊 **Upload Skill** → 選擇下載的 `travel-planner.skill`
5. 安裝完成 ✅

### Step 3：開始使用

在 Claude 對話框輸入任何旅遊相關指令，Skill 會自動觸發：

```
幫我做首爾4天旅遊網頁
整理一下我的巴黎行程
幫我把行程做成網頁
```

---

## 🚀 使用流程

```
你說：「幫我做東京旅遊網頁」
         ↓
Claude 詢問：目的地 / 日期 / 住宿 / 行程 / 機場交通 / 美食清單
         ↓
自動分析：交通票種比價（IC卡 vs Pass vs 散票）
         ↓
生成：完整互動 HTML，另存為 index.html 即可使用
```
